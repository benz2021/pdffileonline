# PDF Annotator 📄

แอปพลิเคชันสำหรับขีดเขียน ไฮไลท์ และเพิ่มข้อความลงในไฟล์ PDF สร้างด้วย Streamlit และ Fabric.js (ผ่าน streamlit-drawable-canvas)

## การใช้งาน
1. อัปโหลดไฟล์ PDF
2. เลือกเครื่องมือที่ Sidebar เพื่อวาดรูป ไฮไลท์ หรือเพิ่มข้อความ
3. กด Commit เพื่อฝังวัตถุลงใน PDF
4. ดาวน์โหลดไฟล์ PDF ที่แก้ไขแล้วกลับไปใช้งาน
